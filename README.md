## 足の８０２０ウォーキングとは

80歳になっても、逢いたい人に自分の足で逢いに行ける社会を目指す市民運動です。

- **協力**：下北沢病院・熱中学園・parkrun Japan

## AMIについて

**AMI**（アミ）は足の８０２０ウォーキングが定める独自の社会貢献参考単位です。

- 1 AMI = 1円換算の参考値
- **AMIはあくまで個人の目安・動機づけのための参考値です**
- 実際の社会保障費削減を保証・約束するものではありません
- 寄付や社会保障制度への直接的な還元を意味するものではありません
- 健康効果を医学的に保証するものではありません
